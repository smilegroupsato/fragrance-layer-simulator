# Fragrance Layer Simulator v0.1

Fragrance Layer Simulator is a small browser-based reading tool for fragrance language. It helps browse fragrance materials, descriptors, molecules, cultural layers, sensory images, reference perfumes, and simple blend structures.

This version does not try to predict smell precisely. The seed data is editorial sample data, not scientific truth.

## Run the App

```bash
npm install
npm run dev
```

Then open `http://localhost:3000`.

## v0.1 Includes

- A glossary of static fragrance entries.
- Detail pages for descriptors, materials, plants, molecules, physiology terms, and image terms.
- A reference perfume browser with separated official information, AI-style layer analysis, public discourse, and user notes.
- A local-state-only perfume memo form.
- A simple blend builder using static TypeScript data.

## v0.1 Intentionally Does Not Include

- Authentication.
- Database storage.
- External APIs.
- Image upload.
- Web scraping.
- Automatic safety judgments.
- Invented official perfume notes.
